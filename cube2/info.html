BananaBread is a port of the Sauerbraten 3D first person shooter (aka the Cube 2 Engine), compiled from C++ and OpenGL to JavaScript and WebGL using Emscripten.

  • Code is zlib licensed and on github
  • Art assets are almost all CC-BY, some are CC-BY-SA and some are CC-BY-NC (see repo for details)

Notes on gameplay:

  • Move with WASD, jump with space, look around with the mouse.
  • Shoot with the mouse, change weapons with 1-5.
  • You can change some settings using the main menu (press `, which is usually under the escape key). For example, add another bot with "add bot", or change the resolution with options->res.
  • 'e' toggles edit mode, which lets you fly around the map and actually modify it.
  • Press 9 to toggle thirdperson mode (to see your own character)

See the troubleshooting page if you trouble running BananaBread in your browser.
